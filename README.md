## AI development needs an operating layer

HELMOR builds local-first tools that help developers and AI agents work with less token waste, less project drift, fewer hallucinations, and stronger safety rails.

```bash
brew tap helmorx/devsuite https://github.com/helmorx/devsuite
brew install helmor
helmor install
helmor dashboard
```

## What we are building

| Project | Purpose | Status |
|---|---|---|
| [DevSuite](https://github.com/helmorx/devsuite) | Local AI development engine with adapters, detectors, modes, and handoffs | v0.1 |
| HELMOR Skills | Product, security, launch, frontend, backend, docs, and workflow skill pack | planned |
| HELMOR Design | Deterministic UI/UX drift checks inspired by modern design audit tools | planned |

## DevSuite pillars

- Token reduction through compact command output, context cards, and graph-first discovery.
- Anti-drift project profiles with truth files, checks, policies, and task state.
- Agent adapters for Codex, Claude Code, Cursor, and Windsurf.
- Local-only detector packs for secrets, unsafe commands, package-runner mismatch, launch risk, and UI drift.
- Observe-first onboarding so existing projects can adopt HELMOR without disruption.
